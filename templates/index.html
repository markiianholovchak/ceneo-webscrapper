What is Scrapify?

Scrapify - web app that allows you to extract opinions about any product from Ceneo.pl. All you need to know is a product's id. We will do the rest for you. We provide you with all neccessary data about the opinions on the chosen product, including analitical charts that will help you decide, whether to buy something or not.

How it was built?

Scrapify was built using such frameworks and libraries as:

  • Flask
  • TailwindCSS
  • ChartJs
  • Pandas
  • Flask table
  • Beautiful Soup

The app is based upon object-oriented programming concepts. What is more, Scrapify uses a local SQLite database to store all the extracted data.
